Fase Dos: Ingenieria de datos

Selección

Se utilizó el archivo Telco-Customer-Churn.csv, el cual contiene información histórica de clientes de una empresa de telecomunicaciones. Este dataset incluye variables demográficas, de uso del servicio, condiciones contractuales y estado de cancelación (Churn).

Limpieza

Se aplicaron las siguientes acciones para asegurar la calidad de los datos:

  • Conversión de tipos: La columna 'TotalCharges', originalmente importada como texto, fue convertida a tipo numérico. Se utilizó errors='coerce' para manejar valores inválidos (como espacios vacíos), convirtiéndolos en NaN.
  • Eliminación de valores faltantes: Se eliminaron las filas con valores nulos, en particular aquellas que resultaron del paso anterior.
  • Eliminación de columnas irrelevantes: Se descartó la columna 'customerID', ya que es un identificador único sin valor predictivo.
  • Codificación de la variable objetivo: La variable 'Churn' fue codificada en formato binario: Yes = 1 y No = 0.

Transformación

  • Codificación de variables categóricas: Se aplicó one-hot encoding a todas las variables categóricas mediante pd.get_dummies, generando un dataframe numérico apto para modelos de machine learning.
  • Normalización: Se normalizaron todas las variables utilizando MinMaxScaler, lo que llevó todos los valores al rango [0, 1].
